qrdev

QR code for your dev server. Scan from your phone, test on any device — instantly.

About

Prepend qrdev to any dev command. It runs your server, watches for a port, finds your local network IP, and shows a QR code you can scan from any device on the same Wi-Fi. No config. No plugins. No framework lock-in. Works with everything.

Way 1 — Run directly

Install globally and prepend qrdev to any dev command.

$ npm i -g qrdev

Then just use it:

qrdev vite qrdev next dev qrdev node server.js qrdev python -m http.server 8000

Way 2 — Add to your scripts

Install as a dev dependency in your project.

$ npm i -D qrdev

Then update your package.json:

"scripts": {
  "dev": "qrdev <your-dev-command>"
}

For example:

"dev": "qrdev vite" "dev": "qrdev next dev" "dev": "qrdev node server.js"

Now npm run dev will automatically show a QR code — no extra steps.
